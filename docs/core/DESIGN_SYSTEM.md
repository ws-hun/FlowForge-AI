# FlowForge AI - Design System

## 1. Design Philosophy

FlowForge design is:

Calm, minimal, and product-focused.

We do NOT design admin panels.

We design a workspace.

---

## 2. Visual Style

- Light Mode First
- Soft contrast
- Large whitespace
- Subtle shadows
- Rounded geometry

---

## 3. Color System

### Background

### Accent
#7C5CFF

### Error

---

## 4. Typography

### Font Family
- Inter (UI)
- JetBrains Mono (Code)

### Rules
- Clear hierarchy
- Generous line height (1.5 - 1.8)
- Avoid dense text blocks

---

## 5. Layout System

- Desktop-first design
- Max width: 1200–1400px
- Grid-based spacing
- Large whitespace is required

---

## 6. Spacing System

Use strict scale:

4px
8px
12px
16px
24px
32px
48px
64px

Never introduce arbitrary spacing.

---

## 7. Border Radius

Single system only:

- Small: 8px
- Default: 12px
- Large: 16px

No other values allowed.

---

## 8. Shadow System

### Soft Shadow
0 4px 20px rgba(0,0,0,0.05)

### Hover Shadow
0 8px 30px rgba(0,0,0,0.08)

---

## 9. Component Principles

All components must be:

- Reusable
- Minimal
- Single responsibility
- Consistent in spacing

---

## 10. Component Style Rules

### Buttons
- One primary action per view
- No excessive variants

### Cards
- Soft elevation
- White background
- Subtle border

### Inputs
- Clean border
- Soft focus state

### Tables
- Avoid unless necessary
- Prefer cards or lists

---

## 11. Animation System

- Duration: 150–250ms
- Ease: ease-out
- Hover scale: 1.01–1.02
- No heavy motion effects

---

## 12. Interaction Principles

- Reduce cognitive load
- Avoid clutter
- Prioritize readability
- Keep UI calm

---

## 13. Anti-Patterns (STRICTLY FORBIDDEN)

- Admin dashboard layout
- Dense tables
- KPI cards
- Data-heavy screens
- Sidebar overload
- Enterprise UI style
